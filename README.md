## Features

- Smooth animations with **Moti**  
- Navigation with **React Navigation**  
- State management with **Redux Toolkit** & **Redux Persist**  
- Local storage with **AsyncStorage**  
- Supports iOS & Android  
- Modern UI components using **Linear Gradient**, **SVG**, and **Lottie**

## Installation

1. **Clone the repository:**

```bash
git clone https://github.com/HUGORDdev/SellifyApp.git
cd SellifyApp
```
2. **Install dependencies:**
```bash
    npm install
    # or
    yarn install
```
3.***Run the app:***
```bash
    # Android
    npm run android

    # iOS
    npm run ios

    # Start Metro server
    npm start
```
